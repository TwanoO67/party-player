## 🚀 Démarrage rapide avec Docker

La façon la plus simple de lancer Party Player:

### Windows
```bash
start.bat
```

### Linux/Mac
```bash
chmod +x start.sh
./start.sh
```

Puis accédez à http://localhost:8080
